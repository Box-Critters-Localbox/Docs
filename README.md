# Box Critters Documentation

Documentation surrounding the internals of Box Critters, a defunct virtual world by RocketSnail games. Here you can find information on types, specifics, and packets. This is a very early work in progress.

Accessable via the wiki tab of this repository.

> All documentation is and will be based off the client version Localbox uses, which is client 161. Down the road, I'd like to switch to using the latest archived version which is client 178.

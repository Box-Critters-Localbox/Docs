# A (player added)

### Description

This packet is fired to all clients in the room when a new player joins the room.

### Parameters

| Key         | Type                                        | Description                         |
| ----------- | ------------------------------------------- | ----------------------------------- |
| playerCrumb | [`PlayerCrumb`](../classes/player-crumb.md) | The player crumb of the new player. |

# R (player left)

### Description

This packet is fired to all clients in the room when a player leaves the room.

### Parameters

| Key         | Type                                        | Description                              |
| ----------- | ------------------------------------------- | ---------------------------------------- |
| playerCrumb | [`PlayerCrumb`](../classes/player-crumb.md) | The player crumb of the player who left. |

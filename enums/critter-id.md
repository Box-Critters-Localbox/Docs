# Critter ID

### Description

Critters are identified via a short ID string as listed below:

### Playable Critters

* `hamster`
* `raccoon`
* `beaver`
* `lizard`
* `penguin`

### Mascots

* `snail`
* `snow_greeter`
* `snowkeeper`
* `snowgirl`
* `snow_patrol`
* `snowgrandma`
